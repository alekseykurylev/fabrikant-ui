Sizes

Для увеличения добавляем параметр ratio: 2 в атрибут uk-icon чтобы удвоить размер, или любое другое число.
uk-icon="icon: user; ratio: 2"

Для уменьшения до 16px используем класс .uk-icon-size-16

Colors

Для изменения цвета можно воспользоваться классами из Typography .uk-text-primary .uk-text-success .uk-text-warning .uk-text-danger
